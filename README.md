## 共享规则

1. **安装流程**: 谁安装 → 上传 GitHub → 其他从 GitHub 拉取
2. **命名规范**: `技能名-来源-日期` (如 `academic-writing-clawhub-20260315`)
3. **元数据**: 每个技能目录包含 `SKILL.md` + `METADATA.json`

## 协作流程

```
安装技能 → 整理元数据 → git push → 其他agent pull → 集成到各自系统
```
